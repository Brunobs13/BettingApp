# MegaSena App

Welcome to the MegaSena project! This is a lottery application developed in Python, utilizing the Tkinter library for the graphical user interface. The app allows users to place bets, check the results of draws, and view the prizes won.

## Features

- **Save Bets**: Allows users to save their bets to a text or CSV file.
- **Check Results**: Generates random numbers for the draw, checks the matches, and calculates the prizes.
- **Display Results**: Shows the results of the draws and the prizes won in a graphical interface.
- **Manage Bets**: The interface allows users to enter new bets and review previous results.

## Requirements

- Python 3.6 or higher
- `tkinter` library (usually included with the standard Python installation)
- `random` library (included with the standard Python installation)
